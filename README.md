# libevent_https_server

#### 介绍
https server with Libevent

#### 软件架构

使用 `Libevent` 实现的服务器，指定 `root` 位置后可以代理静态网站。


#### 安装教程

1.  安装 `cmake`, `libevent`, `fmt`
2.  修改和指定 `SSL` 证书的路径和网站根目录
3.  `cmake ./`
4.  `make`
